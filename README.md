# Linear Liquid Glass Tab Bar

A React Native demo recreating Linear's custom liquid glass tab bar with fluid animations and gesture interactions.

Built with Expo, React Native Reanimated, and Gesture Handler.

## Features

- Liquid glass material effect with blur and gradients
- Animated pill indicator with drag, stretch, and glow
- Custom popup menu with spring animations
- Search input with animated open/close transitions
- Haptic feedback on interactions

## Getting Started

```bash
npm install
npx expo run:ios
```

## Tech Stack

- Expo SDK 55
- React Native 0.83
- React Native Reanimated 4
- React Native Gesture Handler
- Expo Blur / Linear Gradient
